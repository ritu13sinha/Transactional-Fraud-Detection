# Phase 1: Data Exploration and Cleaning

## Objective
To understand the transaction dataset and prepare a clean dataset for further analysis.

## Tasks Performed
- Loaded the transaction dataset.
- Examined dataset dimensions and structure.
- Reviewed column names and data types.
- Generated summary statistics.
- Checked for missing values.
- Checked for duplicate records.
- Converted date columns to datetime format.
- Saved the cleaned dataset.

## Output Files
- Transactional_Fraud_Detection_Phase1.ipynb
- cleaned_transactions.csv

## Status
Phase 1 completed successfully. The cleaned dataset is ready for Exploratory Data Analysis (EDA).
