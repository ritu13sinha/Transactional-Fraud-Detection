# Phase 2: Exploratory Data Analysis (EDA)

## Objective
To analyze the cleaned transaction dataset, identify patterns, trends, and insights, and prepare the data for feature engineering and modeling.

## Tasks Performed
- Univariate Analysis
- Bivariate Analysis
- Transaction Amount Analysis
- Time-Based Analysis
- Customer Behavior Analysis
- Data Visualization
- Pattern Identification

## Deliverables
- EDA Notebook
- Visualizations and Charts
- Observations and Insights

## Key Findings
- Dataset structure and distributions were analyzed.
- Transaction patterns were explored using visualizations.
- Important trends and anomalies were identified.
- Insights were generated for the next phase of feature engineering and modeling.
